Building
	First of all change director to the place where you extracted the source code
	$ make

Source
	src/		source code for the instant messenger
	obj/		object files for the instant messenger
	bin/		im binaries
	Makefile	rules to make the binaries out of the source code
	ReadMe.txt	this readme file, helping you to get started

Class Description
	client		backbone for the client architecture of im
	server		backbone for the server
